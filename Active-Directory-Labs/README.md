# Active Directory & User Management Labs
Lab: Local User & Privilege Management (Windows 11 Pro)
Objective: To demonstrate the ability to manage local user accounts and assign administrative privileges via Computer Management (lusrmgr.msc).

What I did:

Created a new local user account.

Added the user to the Administrators group.

Verified the account could perform elevated tasks.



Skills Shown: User Account Control (UAC), Group Management, Windows 11 Administration.
